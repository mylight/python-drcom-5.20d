链接:校园网客户端设置链接:路由器后台

使用说明:

1.周一到周五的早上校园网允许访问后,如果遇到不能上网现象,请打开"校园网客户端设置"页面，执行"重启客户端"操作.

2.在云插件中心里有定时重启的插件,可以安装该插件后,设定上述重启时间,让路由器定时自动重启,(插件安装需要使用外存储卡或U盘.没有订购的同学,请自行购买实现该功能)

3.云窗视频客户端windows版下载地址,直接将zip格式压缩包解压到电脑桌面,之后找到GYLauncher.exe文件,打开便可24小时使用校园网免费观看视频电影.

4.云窗视频客户端安卓手机版下载地址,地址栏已默认设置为山东理工大学地址(http://210.44.176.242).
